# Event-management-system-Mobile-Application-development-Project-
Event Management App 

This project is a mobile application built with Jetpack Compose that simplifies planning and managing events. The app allows users to create events by entering key details such as title, date, and location. It includes a budget management feature where users can set a budget, track expenses, and view summaries for better financial control. The application leverages a local Room database for persistent data storage and follows MVVM architecture for clean and maintainable code.
